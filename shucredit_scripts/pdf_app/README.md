# 📄 上海大学培养方案PDF解析器 - Web应用版

## ✨ 功能特点

1. **智能PDF解析** - 真正从PDF提取模块和课程信息
2. **预览编辑** - 在导入前查看和修改数据
3. **一键导入** - 直接导入数据库
4. **保持兼容** - 与现有数据库结构完美契合

## 🚀 快速开始

### 1. 启动应用

```bash
cd shucredit_scripts/pdf_app
python run.py
```

### 2. 访问应用

打开浏览器访问：http://localhost:5001

## 📋 使用流程

```
上传PDF → 智能解析 → 预览编辑 → 导入数据库
```

### 步骤1：上传PDF
- 点击或拖拽PDF文件到上传区域
- 支持 `.pdf` 格式

### 步骤2：智能解析
- 自动提取学院和专业名称
- 从PDF解析模块名称和学分
- 提取课程信息

### 步骤3：预览编辑
- 查看解析结果
- 修改学院、专业名称
- 编辑模块名称和学分
- 编辑课程信息

### 步骤4：导入数据库
- 点击「导入数据库」按钮
- 确认数据后完成导入

## 🛠️ 技术栈

- **框架**: Flask 2.x
- **PDF解析**: pdfplumber
- **数据库**: MySQL (pymysql)
- **前端**: HTML5 + CSS3 + JavaScript

## ⚙️ 配置说明

### 数据库配置

在 `app.py` 中修改数据库连接信息：

```python
DB_CONFIG = {
    'host': 'localhost',
    'user': 'root',
    'password': '',
    'database': 'shu_credit',
    'charset': 'utf8mb4'
}
```

### 端口配置

在 `run.py` 中修改端口号：

```python
app.run(host='0.0.0.0', port=5001, debug=True)
```

## 💡 解析规则

### 模块解析
- 支持多种格式：
  - `1. 公共基础课程 67.5学分`
  - `（一）公共基础课程 67.5学分`
  - `公共基础课程（67.5学分）`

### 课程解析
- 识别课程代码：`GBK2000001`
- 自动提取课程名称和学分

### 智能匹配
- 解析的模块会映射到标准结构
- 如果PDF中没有的模块，使用默认值

## 🔧 故障排除

### 问题：无法启动应用

确保已安装依赖：

```bash
pip install flask flask-uploads pdfplumber pymysql
```

### 问题：数据库连接失败

检查数据库配置是否正确，确保MySQL服务正在运行。

### 问题：PDF解析失败

确保PDF文件格式正确，是上海大学标准培养方案格式。
