## 🚀 使用方法

### 启动服务
```bash
# 后端
python app.py

# 前端（新终端）
npm run dev
```

### 填充学院数据
```bash
# 填充通信工程
python populate_colleges.py "通信工程"

# 填充微电子
python populate_colleges.py "微电子科学与工程"

# 填充所有
python populate_colleges.py
```

## 🔧 自定义新学院

在 `populate_colleges.py` 中的 `COLLEGE_DATA` 字典添加新条目：

```python
'新学院名': {
    'code': 'NEW',
    'modules': [
        ('模块名', 学分数, '父模块名'),  # 父模块为 None 表示根模块
        ...
    ],
    'courses': [
        ('课程代码', '课程名', 学分, '所属模块名'),
        ...
    ]
}
```
